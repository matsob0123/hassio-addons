# Changelog

All notable changes to this add-on will be documented in this file.

## [1.0.0] - 2025-01-02

### Added
- Initial release of piSignage Server for Home Assistant
- Full piSignage server functionality port
- MongoDB external database support
- Web interface on port 3000
- Ingress support for secure access
- Multi-architecture support (aarch64, amd64, armhf, armv7, i386)
- FFmpeg video processing
- ImageMagick image processing
- Persistent media storage in /data/media
- Automatic thumbnail generation
- Health check monitoring
- Configurable logging levels
- Session management
- File upload support up to 2GB
- S6-overlay service management
- Graceful shutdown handling

### Configuration Options
- MongoDB URI configuration
- Custom port settings
- Username/password authentication
- piSignage.com username integration
- Maximum upload size configuration
- Session secret customization
- Log level control

### Features
- Player management
- Media asset management
- Playlist creation and scheduling
- Remote display control
- Multi-display support
- Video transcoding
- Image optimization
- License management
- Cloud integration support
